## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/marchiorog/lembreMed.git

# Acesse o diretório do projeto
cd lembreMed

# Instale as dependências
npm install
# ou
yarn install

# Inicie o projeto
npx expo start
# ou, caso não use expo
npx react-native run-android
